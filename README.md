# 🏏 IPL T20 Analytics Dashboard

A comprehensive **IPL Analytics Dashboard** built using **Flask**, **Pandas**, **HTML**, **CSS**, and **JavaScript**. The application provides interactive insights into IPL history through team analytics, player statistics, head-to-head comparisons, championship records, and venue-based analysis.

🌐 **Live Demo:** https://ipl-t20.onrender.com

---

## ✨ Features

### 📊 Dashboard
- Total IPL Matches
- Total Seasons
- Total Teams
- Total Players
- Total Runs
- Total Wickets

### 🏆 Team Analytics
- Matches Played
- Wins & Losses
- Win Percentage
- Total Runs Scored
- Total Wickets Taken
- Venue-wise Performance

### 👤 Player Analytics
- Batting Statistics
- Bowling Statistics
- Fielding Statistics
- Career Performance Metrics

### 🤝 Head-to-Head Analytics
- Team vs Team Comparison
- Win Percentage
- Venue-wise Records
- Highest & Lowest Scores
- Top Batsmen & Bowlers
- Recent Match Results
- AI Match Insights
- Toss Impact Analysis

### 🏅 IPL Champions
- Season-wise Champions
- Runner-up Teams
- Final Match Details
- Player of the Match

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/udayreddyvangala499-collab/IPL-T20.git
```

### Navigate to the project

```bash
cd IPL-T20
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Visit:

```
http://127.0.0.1:5000
```

## ⚡ Performance Optimizations

- Optimized DataFrame loading using `usecols`
- Reduced memory usage from **~520 MB to ~42 MB**
- Cached dataset loading
- Optimized data types using categorical encoding
- Cloud deployment optimized for low-memory environments
